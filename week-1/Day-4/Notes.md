
# Day 4 Notes – Salesforce Flow Builder

# What is Flow Builder?

Flow Builder is a Salesforce automation tool used to create workflows visually without coding.

It helps businesses:
- Automate repetitive tasks
- Update records automatically
- Send emails and notifications
- Improve efficiency

---

# Why Businesses Use Automation

Businesses automate processes because:
- Manual work takes time
- Human errors occur frequently
- Repetitive tasks reduce productivity

Automation provides:
- Faster execution
- Better consistency
- Improved customer experience
- Reduced operational cost

---

# Types of Salesforce Flows

## 1. Screen Flow
- Requires user interaction
- Uses forms/screens
- Example: Student registration form

## 2. Record Triggered Flow
- Runs automatically
- Triggered by record creation/update
- Example: Auto email after registration

## 3. Scheduled Flow
- Runs at fixed times
- Example: Fee reminder emails

## 4. Autolaunched Flow
- Runs in background
- No screens involved

---

- Create records

## Decisions and Conditions
Flows can make decisions using:
- If conditions
- Logic branches
